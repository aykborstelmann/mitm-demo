# Man In The Middle - DEMO Website

## Getting started

### Requirements
 * Python3
 * pip
 * virtualenv
 * flask

### TLDR
````shell script
pip3 -m virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
./prepare.sh
````

### Startup Website
````shell script
flask run
````

or:

````shell script
python3 -m flask run
````
